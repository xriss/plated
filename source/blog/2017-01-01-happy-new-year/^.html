As it is I think we have enough bits working now that this can replace the static site generation in d-portal, which is where this JavaScript code came from originally. Right now we are also available on NPM but probably a good idea to wait a little while for things to settle down as everything is still in a state of flux design wise as I try and take the design lessons from the old code and attempt to unify it into something simple.
